1. 請整理課堂上所提的Git指令，顯示其語法語給予明顯的例子

Ans:
```
git clone（git上的ssh) #把github上倉庫的檔案下載到本地倉庫

git add 檔名(或 . #全部檔案) #添加至暫存區

git commit #從暫存區提交到本地倉庫 可以進行備註

git push #從本地倉庫推送到git倉庫

git fetch #從遠端倉庫下載所有新的資料，但不會自動合併到本地分支

git merge (分支) #所在位置合併輸入的分支

git pull #從遠端倉庫下載新資料並自動合併到當前分支

git branch (新的分支名稱)#用於創建新分支(用於做新東西而不影響之前版本)

git checkout (分支名稱)#切換分支
```

Ans:
```
1.每周可能4小時以上，時間會用在複習課堂上所記錄的筆記與作業上，若作業沒在時間內做完可能就無限增加時間...

ChatGpt:
```
